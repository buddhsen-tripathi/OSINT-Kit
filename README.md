## What's inside

The full directory lives on the [site](https://buddhsen-tripathi.github.io/OSINT-Kit/), grouped into:

- OSINT Aggregator Tools
- Metadata & File Analysis
- Image / Video / Media Analysis
- Data Breach & Leak Search
- Location Tracking
- Domain Information
- Social Media Intelligence (SOCMINT)
- Marketplaces
- Operating Systems
- PGP Key Servers
- OSINT APIs
- Browser Extensions for OSINT
- Recording & Auditing
- Datasets & Public Records
- Learning Resources
- Miscellaneous

## Tech stack

The site is built with [Astro](https://astro.build/) + [Starlight](https://starlight.astro.build/) and deployed to GitHub Pages via GitHub Actions.

## Local development

```bash
npm install
npm run dev    # http://localhost:4321/OSINT-Kit/
npm run build  # output to ./dist
```

Tool entries live in [`src/content/docs/index.md`](src/content/docs/index.md). Edit there, push to `main`, and the workflow rebuilds and redeploys.
